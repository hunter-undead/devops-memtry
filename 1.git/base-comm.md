# Как сделать свой репозиторий из локальной папки на ПК.
1) На GitHub создаем репозиторий (сетевой).
2) Открываем GitBash - переходим в нужную нам папку нам ПК, которая будет локальным репозиторием:
- git init (в директории)
- git add . (добавит все файлы, что тут есть в индекс = позволит запушить в сетевой репо) / git add (перечисляем, что хотим добавить)
